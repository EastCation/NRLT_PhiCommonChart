# BPM

本页介绍谱面单个BPM组成部分。
::: info 提示
**本页所有字段兼容等级为 0。**
:::

## Json直观示例

```json
{
  "Bpm": 120.0,
  "StartBeat": 0.0
}
```

## 结构规范

| 唯一标识符 |    字段名    |  类型   | 描述             |  默认值  | 加入版本 |
|:-----:|:---------:|:-----:|:---------------|:-----:|:----:|
|   1   |    Bpm    | float | 段落BPM值，单位为 BPM | 120.0 |  1   |
|   2   | StartBeat | float | BPM开始的拍数       |  0.0  |  1   |
